# قالب ZTM SmartBuilding (Apple-like Pro)

نسخه 1.3.0  
- هرو تمام‌صفحه با ویدئوی سینمایی  
- سکشن‌های ویژه تمام‌عرض (Bands)  
- پارالاکس نرم با GSAP/ScrollTrigger  
- Sticky Sub-Nav در صفحات محصول و پروژه  
- کارت‌های Explore شبیه اپل  
- پنل تنظیمات کامل در پیشخوان (بدون ACF)

## نصب
1) فایل zip را از نمایش → پوسته‌ها → افزودن → بارگذاری نصب کنید.  
2) قالب را فعال کنید.  
3) به مسیر **تنظیمات قالب زرین** در منوی پیشخوان بروید و تنظیمات را انجام دهید.

## تنظیمات
- رنگ‌ها و هویت بصری
- هرو تمام صفحه (تیتر/زیرتیتر/ویدئو/CTA)
- Bands اپل‌مانند (۳ سکشن ویژه)
- Explore Cards (۴ کارت)
- اطلاعات تماس

## محصولات
CPT: محصولات  
متاباکس داخلی:
- قیمت
- موجودی
- لینک PDF
- مشخصات فنی (هر خط: عنوان | مقدار)

## پروژه‌ها
CPT: پروژه‌ها  
متاباکس داخلی:
- کارفرما
- موقعیت
- تاریخ اجرا
- خدمات انجام شده
